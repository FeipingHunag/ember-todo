Ember Todo: An rails4 + ember1.0 + ember-data1.0 demo
==================================
- [no-coffeescript branch](https://github.com/FeipingHunag/ember-todo/tree/no-coffeescript)
- Live demo <http://ember-todo.herokuapp.com>

USAGE
=====

Run:

    rake migrate
    rake db:seed
    rails server

Visit the webserver at:

    http://localhost:3000
